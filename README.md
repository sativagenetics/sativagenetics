## Core focus

I am interested in AI systems that leave the chat window and operate inside real workflows.

- Local-first AI agents and command centers
- Realtime dashboards for business owners and operators
- Restaurant tech: stock, reviews, customer messages, sales signals
- Privacy-aware automation with clear audit trails
- Windows / Android / PWA-first product development
- Practical prototypes that can become paid products

## How I like to build

```text
Observe       -> make the hidden operational state visible
Control       -> give the owner a simple dashboard and safe actions
Automate      -> automate only after the workflow is measurable
Verify        -> add local QA, smoke tests and evidence before scaling
Ship          -> prefer working prototypes over slideware
```

## Product principles

- Build useful tools, not demos
- Make operations visible before automating them
- Prefer local-first and privacy-aware architecture
- Keep the operator in control with clear logs, state and rollback paths
- Design for business owners first, developers second
- Validate with working prototypes, QA evidence and measurable outcomes

## Current signal map

```text
Local-first systems   -> privacy-aware software that runs close to the user

Product automation    -> replacing repetitive admin work with reliable flows
```
