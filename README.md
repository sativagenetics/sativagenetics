# SATIVA

I build local-first AI agents, real-time automation dashboards, and practical software systems for small businesses.

My current focus is turning everyday business operations into visible, controllable systems: restaurant workflows, owner dashboards, customer messages, stock signals, accounting summaries, review monitoring, and autonomous task execution.

## What I am building

### SATIVA Restaurant Agent
An AI-powered restaurant operations layer for owners and managers.

- Sales, tables, stock and purchasing visibility
- Cancellation, loss and issue reporting
- Google review and customer-message workflows
- Menu-aware assistant behavior
- Phone-visible control and summaries

### SATIVA Control Center
A local-first command center for monitoring projects, jobs, ports, logs, alerts, system resources and AI agent activity in real time.

- Live telemetry and server-sent events
- Local-only dashboard architecture
- Project and service monitoring
- AI workflow visibility

### KROIRA IPTV
Cross-platform IPTV products for Android and Windows with a focus on stable playback, clean UX and practical media workflows.
